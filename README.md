# Unichat

## Descrizione
```
Si è voluto realizzare, mediante l’uso di VUE, SASS e PHP una web app simile ad un social network, dove avremo la sezione di login, visualizzazione dei post e dei commenti relativi, visualizzazione della messaggistica e visualizzazione della propria pagina personale; l’obbiettivo principale è quello di mostrare come, grazie all’utilizzo del framework VUEJS è possibile realizzare una Single Page Application, in maniera semplice e funzionale.
